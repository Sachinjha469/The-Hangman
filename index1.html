INSTRUCTIONS FOR PLAYING HANGMAN

  1. Play individually or in groups.
  2. The game is all about guessing
    1. Select a letter of the alphabet from the listed alphabet buttons provided to you
    2. If the alphabets you clicked matches the answer then VOILA you are about to save the man!
    3. If the alphabet you clicked doesn't fills up the blank spaces then don't loose hope you still have several attempts left!
    4. Even If you have lost the game, you still have some guesses left to know the correct answer but sadly you have killed the Man!
  3. So let's GET STARTED!
